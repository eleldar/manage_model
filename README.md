# Модель оптимального управления

## Installation
```
$ python3 -m venv venv
$ . venv/bin/activate
(venv)$ pip install -r requirements.txt
```

## Using
```
### first terminal
(venv)$ python main.py
### second terminal 
(venv)$ python client.py
```

Then open browser and sent `127.0.0.1:8050`,
where change number to predict days.
